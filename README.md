## What Is This

`easeus-skills` is a skill routing repository.
Describe your symptoms and the skill routes the problem to the right workflow, such as disk health diagnosis, boot repair, migration validation, partition conversion, and write-protection removal.

## Quick Start

### I Am an Agent

```text
For AI:
Read https://github.com/wrcopilot-org/easeus-skills/raw/refs/heads/main/SKILL.md and follow the instructions to check my disk health.
```

### I Am a Human

Quick install:

```bash
npx skills add https://github.com/wrcopilot-org/easeus-skills -y -g
```

Claude Code: "Use this Skill to check my disk health."
